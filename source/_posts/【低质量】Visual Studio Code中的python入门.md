# python版本

Python2和Python3两种，但是作为学习着重于新的Python3。

## 检查版本

win+r  输入cmd 打开终端，键入以下命令

```bash
python --version
```

```bash
py --version
```

如果正常的话是会有Python版本号的(以3开头即为Python3)，如果报错则说明你的电脑上没有安装Python，或者安装了但是没有添加到环境变量里。

## 没有则安装

去官网即可，直接点击下载。

[Download Python | Python.org](https://www.python.org/downloads/)
